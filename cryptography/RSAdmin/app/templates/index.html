Information

The lead developer of this public key database is Bob. Bob is also a budding cryptographer and so he thought of using message signatures to verify who is an admin. Bob follows PKCS v1.5 and chose to use a strong hash function, SHA256, to hash the message I am an administrator.. The following psuedo-code is used to verify signatures. 

message = "I am an administrator."

verify_signature(signature):
  publickeys = load_json("keylist.json")

  for key in publickeys:
    if verify(hash(message), b64decode(signature)):
      return True

  return False
